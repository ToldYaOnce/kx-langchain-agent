================================================================================
GOAL CONFIGURATION TEMPLATES FOR AI AGENT SYSTEM
================================================================================

This file contains pre-built goal configuration templates for different 
industries. Each template provides a complete conversation flow optimized 
for that industry's typical customer journey.

================================================================================
TEMPLATE STRUCTURE
================================================================================

Each template contains:
- label: Industry name
- description: What type of business it's for
- goals: Array of goal objects with the following fields:
  - id: Unique identifier
  - label: UI-friendly name
  - type: data_collection | scheduling | conversation
  - order: Sequence number (1, 2, 3...)
  - priority: critical | high | medium | low
  - purpose: What this goal achieves
  - adherence: 1-10 scale (how persistent the agent should be)
  - triggers: When this goal activates
  - dataToCapture: What information to collect
  - behavior: How the agent pursues this goal
  - actions: Events to trigger on completion (optional)
